VLSI Logic Design with Cadence

Project Overview

This project involved designing, simulating, and verifying digital logic components using Cadence Virtuoso. The objective was to implement logic gates, adders, and their corresponding layouts while ensuring manufacturability through rigorous design checks and simulation.

Schematic Design

  • Constructed transistor-level schematics for AND, OR, XOR, NAND gates.
  • Designed hierarchical components including half-adders and full-adders.
  • Prepared clean and functional netlists for layout integration.

Layout Design

  • Developed full-custom layouts from schematic designs using Cadence Layout Editor.
  • Ensured minimal area usage and adhered to best practices for layout symmetry and routing.
  • Utilized all standard layers (poly, metal, diffusion) and ensured connectivity across instances.

Verification and Simulation

  • Performed DRC (Design Rule Check) to verify layout compliance with manufacturing rules.
  • Conducted LVS (Layout Versus Schematic) checks to validate the physical layout against the schematic netlist.
  • Simulated gate and adder circuits using Spectre to assess functional correctness and timing behavior.
  • Measured propagation delays across varying voltage levels; worst-case delay for the 4-bit adder was approximately 128.11 ps.

Conclusion

The project provided valuable experience in the end-to-end design and verification flow for custom digital logic. Working through both schematic and layout processes reinforced an understanding of VLSI design principles, while the simulation and validation stages ensured functional and physical correctness.
