Digital Logic Design with Cadence

This project involved the complete custom design flow of digital logic circuits using the Cadence Virtuoso suite. I developed schematics and corresponding layouts for foundational digital components and verified them using industry-standard checks.

  • Designed schematics for logic gates (AND, OR, XOR, NAND) and combinational logic circuits (half-adder, full-adder).
  • Created reusable symbols and built hierarchical blocks to integrate multiple modules.
  • Generated full custom layouts for each circuit in Cadence layout editor, optimizing for area and minimizing parasitics.
  • Ran Design Rule Check (DRC) to ensure physical manufacturability.
  • Performed Layout Versus Schematic (LVS) verification to confirm layout fidelity to circuit function.
  • Used Spectre simulations to validate logic behavior, propagation delay, and power characteristics.

This project emphasized not only accurate logic design but also the real-world constraints of IC fabrication, verifying the designs at both the schematic and physical levels.
