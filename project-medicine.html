Automated Pill Dispenser

Project Overview

This device automates the dispensing of medication using a servo-controlled mechanism and verifies dispensing through a photoresistor sensor. An LCD display communicates messages to the user, while a shaker module alerts them to dispense. It uses an Arduino Uno and basic control logic implemented in C++ using hardware timers, loops, and conditional triggers.

Hardware & Mechanical Design

The housing was modeled in CAD and 3D printed. It consists of modular red and orange parts to hold the electronics, pill chamber, and moving components securely. A central servo arm pushes pills out when triggered.

Embedded Control Logic

  • Controlled a servo motor to push pills on a timed schedule using `millis()`-based loops.
  • Triggered a vibration module to shake pills into the correct spot for dispensing.
  • Used a photoresistor to detect if a pill was successfully dispensed (light interruption detection).
  • Displayed system status and instructions on a 16x2 LCD screen.
  • Implemented logic to retry dispense if pill was not detected within a threshold delay.

Key Features

  • Fully automated, timed pill dispensing process with fallback logic
  • Modular and printable mechanical design for ease of assembly
  • Arduino Uno-based system with loop timing and interrupt-free logic
  • Live feedback using LCD

Conclusion

This project demonstrates a low-cost, scalable solution for elderly or chronically ill patients to manage medication independently. The integration of sensing, actuation, and user interface showcases real-world embedded system design and mechanical prototyping.
